# Моделирование логического элемента И-НЕ

В данном проекте реализован алгоритм анализа динамических процессов в электронных схемах методом переменных состояния

![Схема](images/Scheme1.png)

## О компоненте типа "G"

Компонент, обозначенный в программе **G**, представляет собой **источник тока, управляемый напряжением**

В контексте моделирования МДП-транзисторов этот элемент описывает зависимость тока стока от напряжения на затворе. Ток, генерируемый данным источником, не является фиксированной величиной, а вычисляется линейно в каждый момент времени по формуле:

$$I = S \cdot U_{вх}$$

где:
*   $S$ - крутизна характеристики
*   $U_{вх}$ - управляющее входное напряжение

### Параметры симуляции

После построения модели программа запросит начальные условия. Так как мы моделируем переключение схемы, я предполагаю, что выходная емкость изначально заряжена до уровня питания

1. Для переменных состояния, связанных с выходом (например, **U_Csi2** или **U_Csi3**), введите значение **5**.
2. Для остальных переменных можно оставить значение по умолчанию (0).

При выборе переменных для наблюдения выберите **U_Cn** (напряжение на нагрузке)
